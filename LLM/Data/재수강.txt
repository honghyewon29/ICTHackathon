■ 재수강

● 재수강 소개

- 기 이수한 과목의 성적보다 높은 성적을 취득하기 위한 수강제도

● 재수강 조건

- 기존 취득성적이 C+ ~ F 인 경우 신청가능 (재수강 과목의 성적은 최대 B+로 제한 합니다.)

● 재수강 방법

- 인터넷(모바일) 수강신청 시 과목명이 동일하거나 학수번호가 동일한 과목을 수강신청하게 되면 재수강을 확인 할 수 있는 표시가 됨.

● 재수강 처리

- 공시성적기간 후 성적집계처리 작업을 거쳐 자동 성적비교하여 낮은 성적은 자동 삭제됨.
- 기이수한 과목과 재수강과목의 성적이 동일할 경우 재수강 학기의 성적이 삭제 됩니다.

● 교양선택 수강신청 시 유의과목

- 선택교양과목 중 글로벌교양 1~6, 생활구기, 라켓스포츠, 전통무예 등은 과목명 뒤에 ( )안에 추가명이 표기 됩니다.
                                    ( )안의 추가명이 상이해도 과목명이 같을 경우 다시 수강신청시 재수강 처리가 됩니다.
- 글로벌교양 1~6은 숫자 까지가 과목명입니다. 수강 신청 시 이점 유의하시기 바랍니다.
                                    예 ) 2010-1:전통무예(검도) / 2011-1:전통무예(유도) -재수강 
                                    2018-1:글로벌교양 3(신화와 철학) / 2018-2:글로벌교양 3(역사인류학)-재수강
